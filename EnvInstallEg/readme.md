记录我安装环境的一些过程，有的时候要在本机测试没问题，再去服务器上部署。

## 常见的通用错误

使用代理后，用anaconda创建环境报SSL错误，因为代理服务器会影响与Conda源的通信，导致无法握手。
关闭代理即可。
